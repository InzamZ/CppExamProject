## 基本功能和实现方法
### 储存
使用双向链表进行储存,可以参考课程第四章的代码.实现代码在`link_list.h`中.主要以下功能:
 - [ ] 插入元素(按照升序或降序)
 - [ ] 删除元素
 - [ ] 排序功能(考虑冒泡排序)
 - [ ] 文件导入数据(建立在插入排序上)
 - [ ] 正向反向迭代器

 ### 类:成员(employee)
 即数据字典,人员信息包括:
 - 字符串 : `员工号`,`姓名`,`性别`,`出生日期`,`入职日期`
 - 浮点数 : `年薪`
 - 类属性 : `部门`,`职位`

### 类:结果集
查询等结果要求返回一个集合,需要定义相关类.
